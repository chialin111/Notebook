Membranous Nephropathy (MN) Chapter 3

免疫複合體沉積於腎小球基底膜上皮下，導致腎病症候群。最常見於中老年男性。

原發性 ≈ 80% 抗體媒介 次發性需排除
🔬 診斷重點
抗 PLA2R 抗體陽性 + 腎病症候群→ 可不需腎臟切片即診斷 MN（敏感性 78%、特異性 99%）
抗 PLA2R 陰性→ 仍需切片，並染色確認腎組織 PLA2R 抗原沉積
無論抗體狀態，所有 MN 患者均需評估次發性原因（惡性腫瘤、HBV/HCV/HIV、NSAIDs、自體免疫疾病）
次發性抗體：抗 THSD7A 抗體（較罕見，常與惡性腫瘤相關）
📊 疾病進展風險分層
風險蛋白尿eGFR白蛋白抗PLA2R
低 <3.5 g/d >60 >30 g/L 低效價
中 3.5–8 g/d >6個月 正常 25–30 中效價
高 >8 g/d >6個月 60–30 <25 高效價
極高 任何 <30 或快速下降 <20 持續高

⚠️ 抗 PLA2R 抗體消失早於蛋白尿緩解，應以免疫學緩解為療效指標。

💊 治療策略
⚠️ 不需免疫抑制的條件

蛋白尿 <3.5 g/d、白蛋白 >30 g/L（BCP法）、eGFR >60 → 保守治療即可

建議（1B）

有≥1項進展風險因子者：Rituximab 或 Cyclophosphamide + 隔月 GC × 6個月 或 CNI ≥6個月

各治療比較

Rituximab 優先 中等風險以上首選；副作用少
CNI（CsA / Tacrolimus） 中等風險可用；高復發率
Cyclophosphamide + GC 高/極高風險；累積劑量 ≤36g
MMF 單藥 不建議（無足夠緩解率）
📈 免疫學監測（治療後6個月）
免疫學緩解
抗PLA2R <2 RU/mL → 停止或不增加治療
部分反應
效價下降但仍陽性 → 觀察或繼續治療
無反應 / 持續高效價
更換治療方案；考慮再切片確認
⚠️ 特殊情境
血栓栓塞預防：白蛋白 <25g/L（BCG法）者高風險，需評估抗凝治療（VTE vs. ATE 分別評估）
腎移植後復發：移植前抗PLA2R陽性→高復發風險；移植後定期監測抗體≥6–12個月
抵抗性疾病：排除合併繼發性FSGS（白蛋白正常化但蛋白尿持續）；難治者轉介專門中心
Minimal Change Disease (MCD) Chapter 5（成人）

足細胞病變（podocytopathy），以 T 細胞失調為主要機轉，佔成人腎病症候群 10–25%。腎臟預後良好，但復發率高。

類固醇敏感 >80% 復發率高 類固醇抵抗 10–20%
🔬 診斷要點
成人 MCD 必須靠腎臟切片確診（無法從臨床資料單獨診斷）
光學顯微鏡幾乎正常或輕微系膜增生；電子顯微鏡見廣泛足突消失，無電子緻密沉積
需排除次發性：Hodgkin's lymphoma、Lithium、NSAIDs
注意：切片量不足可能遺漏早期 FSGS 病灶
🏥 初次治療 建議（1C）
第一線

高劑量口服類固醇：Prednisone 1 mg/kg/d（最高80mg），達完全緩解後2週開始減量，總療程約24週

最長使用16週；50%患者於4週緩解，另10–25%在16週內緩解
成人反應速度比兒童慢，需給予足夠時間
類固醇禁忌時替代方案：Cyclophosphamide、CNI、MMF
🔄 定義（緩解 / 復發 / 抵抗 / 依賴）
完全緩解 PCR <200 mg/g（<0.2 g/d）
復發 緩解後 PCR ≥2000 mg/g（≥2 g/d）
頻繁復發（FR） 6個月內 ≥2次，或12個月內 ≥3次
類固醇依賴（SD） 減量或停藥14天內復發 × 2次
類固醇抵抗（SR） 高劑量 GC 16週仍未緩解
💊 FR / SD MCD 治療選項 建議（1C）
Cyclophosphamide — 傳統首選第二線，緩解持久；注意不孕風險，單療程 <12週
Rituximab — 緩解率 65–100%，顯著減少復發次數；成本高但給藥方便（單療程）
CNI（CsA / Tac） — 緩解率 70–90%；但停藥後復發率高，需長期維持
MPAA（MMF / 麥考酚酸） — 緩解率 65–85%；適合 Cyclo + CNI 後仍復發者
臨床決策考量

育齡期 → 避免 Cyclophosphamide；偏好單次給藥 → Rituximab；成本考量 → Cyclophosphamide；長期依賴 → CNI 但需監測腎毒性

Focal Segmental Glomerulosclerosis (FSGS) Chapter 6

KDIGO 2021 對 FSGS 命名提出重大修訂，強調分類影響治療決策。

原發性 FSGS 繼發性 FSGS 基因性 FSGS FSGS-UC（未定原因）
📋 KDIGO 2021 分類定義
原發性 FSGS 需免疫治療
切片有 FSGS 病灶 + EM 廣泛足突消失 + 腎病症候群（蛋白尿>3.5g/d + 低白蛋白）
繼發性 FSGS 不需免疫治療
已知病因（肥胖、HIV、APOL1、CNI、mTOR抑制劑、SARS-CoV-2等）
FSGS-UC 不需免疫治療
無腎病症候群、無廣泛足突消失、無可識別病因
基因性 FSGS 不需免疫治療
足細胞或基底膜蛋白基因突變（家族性 / 年輕 / 難治者優先檢測）
⚡ 診斷與鑑別要點
無腎病症候群的 FSGS → 先排除繼發性原因，勿直接給予免疫治療
EM 廣泛足突消失（difuse FPE）是原發性 FSGS 必要條件，但部分原發性患者 FPE 可低至 30%
組織型（NOS / Tip / Cellular / Perihilar / Collapsing）鑑別價值有限，不能區分原發 vs 繼發
基因檢測適應症：年輕、家族史、染色體異常、對類固醇完全抵抗 → 建議轉介專門中心
💊 原發性 FSGS 治療 建議（1D）
第一線

高劑量口服類固醇：Prednisone 1 mg/kg/d（最高80mg），達完全緩解為止（最長16週），緩解後維持類固醇≥6個月

類固醇相對禁忌 → 替代選擇：CNI（CsA 或 Tacrolimus）
建議（1C）— 類固醇抵抗性原發性 FSGS

Cyclosporine 或 Tacrolimus ≥6個月，有效者繼續維持≥12個月以降低復發

🔄 定義（FSGS）
完全緩解 PCR <200 mg/g
部分緩解 蛋白尿下降≥50% 且 PCR 200–3500 mg/g
類固醇抵抗 高劑量 GC 16週後蛋白尿未達部分緩解
CNI抵抗/不耐受 → 轉介專門中心，考慮再切片或臨床試驗
⚠️ 特殊情境
腎移植後復發：原發性 FSGS 復發率高（32%，中位數1.5個月）；基因性 FSGS 移植後幾乎不復發
復發 FSGS（前次類固醇敏感）：治療方式同 FR/SD MCD
基因性 FSGS：免疫治療抵抗為常見特徵，確認基因突變後應停止免疫治療，評估移植時機

監測生物標記 蛋白尿 / PCR 蛋白尿 / PCR 抗PLA2R抗體效價 + 蛋白尿
腎臟長期預後 極佳（類固醇敏感者） 中等（抵抗者差） 中等（40%進展）

兩者均有廣泛足突消失，可能代表同一疾病譜系

📌 臨床關鍵記憶點
MN：抗 PLA2R 抗體是診斷與監測雙重工具；免疫學緩解（抗體轉陰）早於臨床緩解
MCD：腎預後極佳但高復發率；成人反應比兒童慢，給予足夠時間（最長16週）
FSGS：分類決定治療！繼發性/基因性/FSGS-UC 均不需免疫治療
Cyclophosphamide 累積劑量上限：一般 ≤36g；保留生育能力 ≤10g
所有腎病症候群：均需最佳化支持治療（RAS 抑制劑、血壓控制、水腫管理）
